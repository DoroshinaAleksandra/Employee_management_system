# Employee management system
This program is an employee management system 
with a web-interface. Modern companies manage their employees through centralized 
systems that streamline HR operations. 
This application addresses key business needs 
by providing essential CRUD-operations for 
everyday HR tasks without unnecessary complexity. 
The web-based interface allows users to access the system from any device 
with a web browser, eliminating the need for installation on client machines.

## Key Features

- **CRUD-Operations**: create, read, update, and delete employee records
- **Web Interface**: intuitive interface built with NiceGUI
- **Data Validation**: Pydantic schemas for input validation
- **Database**: made with SQLAlchemy library
- **Multi-Currency Support**: RUB, USD, EUR, GBP, CNY, JPY, KZT, BYN
- **Timezone Support**: UTC-12 to UTC+12
- **Date Management**: date of birth and hire date tracking

## **Installation**
1. Clone the repository   

    ``` 
    git clone https://github.com/DoroshinaAleksandra/Employee_management_system   
    cd your-repo
   ```
2. Install dependencies
   ```
   pip install -r requirements.txt
   ```
3. Run the application
   ```
   python -m src.management_system.main
   ```
